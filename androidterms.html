Android Application Privacy Policy

The MeshCentral Android application does collect or send any information back to the application authors or to meshcentral.com. This is an open source application, the source code is here. When installing the Android application, the user or that application must pair it to a MeshCentral server using a URL or QR code and that server will receive connections from the Android application. All information exchanged is only between the Android application and the MeshCentral server the application is paired with.
